GTAWorld — local

Double‑click Play.bat in the project folder: if a packaged GTAWorld.exe exists it runs; otherwise you get a short dialog with the real options (Unreal cannot ship as a zero‑install click without that build).

Open the Unreal project (Windows)

Do not click file:/// links from inside Cursor — they can open as a broken path (c:%5C…). Use one of these instead:

  • Windows File Explorer: double‑click GTAWorld.uproject or Open-GTAWorld-Local.bat in this folder.
  • Cursor: Terminal → Run Task… → Open GTAWorld in Unreal Editor.
  • To edit the .uproject JSON in Cursor: Ctrl+P, type GTAWorld.uproject, Enter (opens from the workspace, not a chat link).
After UE opens
  1. Let C++ compile on first open if prompted.
  2. File → New Level → Empty Level, save as Content/Maps/StartupMap.
  3. Add a Player Start, light, and floor with collision; then in Config/DefaultEngine.ini uncomment GameDefaultMap / EditorStartupMap (see comments in that file).
  4. Press Play in the Editor to test locally (PIE).

Serve this folder with: python -m http.server 8765 — then open http://127.0.0.1:8765/local.html
